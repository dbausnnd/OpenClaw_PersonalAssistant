# ObsidianLiveSync

Dieses Repository-/Projektsegment dokumentiert die geplante Einrichtung von Obsidian LiveSync auf dem VPS.

## Ziel

- Gemeinsamer, bidirektionaler Sync für Dirks Obsidian-Vaults
- VPS als zentrale Sync-Infrastruktur
- Assistent kann auf Notizen zugreifen und Notizen anlegen
- Alle Änderungen und Einstellungen nachvollziehbar dokumentieren
- Rückbau muss jederzeit möglich sein

## Grundprinzip

Vor jeder technischen Änderung werden dokumentiert:
- aktueller Zustand
- Zielzustand
- konkrete Konfiguration
- Rückbaupfad
- betroffene Dateien, Dienste, Ports und Verzeichnisse

## Struktur

- `docs/` — Ablauf, Architektur, Entscheidungslogik
- `config/` — geplante oder tatsächlich verwendete Konfigurationsdateien
- `backups/` — Sicherungen von Konfigurationen vor Änderungen
- `rollback/` — Rückbauanleitungen und Soll-Ist-Vergleich
